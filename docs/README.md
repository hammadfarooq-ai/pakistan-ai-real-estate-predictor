## Quick Start

1. Copy environment files:
   - `backend/.env.example` to `backend/.env`
   - `frontend/.env.example` to `frontend/.env`
2. Start the stack:
   - `docker compose up --build`
3. Open:
   - Frontend: `http://localhost:3000`
   - Backend API docs: `http://localhost:8000/docs`
